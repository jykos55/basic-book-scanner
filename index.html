Book Nook keeps your list in a Google Sheet you control — nothing is stored on a server of ours. Set one up once (a couple of minutes), then paste its Web App URL below.

  1. Create a blank Google Sheet.
  2. Extensions → Apps Script, delete the starter code, and paste in the script below.
  3. Deploy → New deployment → type "Web app" → Execute as Me, access Anyone with the link → Deploy.
  4. Copy the Web App URL it gives you and paste it in below.
